Trading Anomaly Tracker

Description: The visualizations below were produced in conjunction with my MSc thesis, which involved working with a major financial institution in order to find a viable method of using technology to automatically identify indications of rogue trading in big data. This mainly involved exploring machine learning technologies using small subsets of a database in order to test whether normal trader behaviour could be learned. One of the methods explored, which invovled using Decision Tree based learners, such as a Random Forest learner or the xGBoost learner gave promising indications of being a viable technology for the financial investigative team to utilize. This dashboard below is a modfied version of the framework used to present the results to the client. The data has been faked to for client confidentiality.
